  1. 핵심 정의 (Core Identity)

  SINABRO는 단순한 독서 SNS가 아니다.
  이 플랫폼은 사용자의 “읽은 책”이 아니라,
  👉 “사고 방식, 질문 수준, 가치관의 변화”를 기록하고 드러내는 지적 정체성 네트워크다.

  사용자는 책을 소비하는 사람이 아니라
  👉 “생각을 생산하는 사람”으로 존재한다.

  2. 문제 정의 (Problem)

  기존 독서 플랫폼 (Goodreads 등)은 다음 한계를 가진다:

  읽었는지 / 별점 / 리뷰 → 표면적 데이터만 존재
  사고 변화, 질문, 관점 충돌 → 기록되지 않음
  사용자 간 상호작용 → 공감(좋아요) 중심, 깊이 없음
  결과적으로 → 지적 수준이나 사고 구조가 드러나지 않음
  3. 해결 방향 (Solution)

  SINABRO는 3가지 축으로 문제를 해결한다:

  ① Intellectual Status Network (지적 정체성 네트워크)

  핵심 질문:
  👉 “이 사람은 어떤 생각을 하는 인간인가?”

  기능:
  📘 “이 책에서 바뀐 생각” 피드
  ❓ “내가 던진 질문” 기록
  🔥 “논쟁 기반 댓글 시스템”
  좋아요 ❌
  반박 / 확장 / 재해석 ⭕
  🧠 독서 수준 레벨링
  철학 / 문학 / 과학 / 사유 깊이 기반
  구조:
  사용자는 책이 아니라
  👉 “자신의 사고 패턴”을 전시한다
  ② AI Taste Builder (지적 성장 설계 시스템)

  기존 추천:
  → “비슷한 책 추천”

  SINABRO:
  → “너를 깨뜨리는 책 추천”

  AI 역할:
  🧩 사고 패턴 분석
  ⚠️ 편향 탐지
  🔄 반대 관점 주입
  예시:
  “너는 효율 중심 사고가 강함 → 인간성/철학 부족”
  “이 책은 너의 가치관과 충돌함 → 읽어야 함”
  핵심:

  👉 추천이 아니라
  👉 “지적 성장 경로 설계”

  ③ Literary Social Layer (문학 기반 SNS 경험)

  SINABRO는 기존 SNS 구조를 가져오되
  문학적으로 재해석한다

  기능:
  📱 피드 (생각 중심)
  🔍 Explore (장르 기반 탐색)
  🎭 감정 기반 리액션 (단순 좋아요 확장)
  🧾 짧은 글 / 긴 글 모두 가능
  중요한 규칙:
  모든 콘텐츠는 문학/사고 기반
  광고, 잡담, 일반 SNS 요소 → 배제
  4. UX 철학 (Critical Design Philosophy)

  SINABRO는 다음을 절대적으로 지향한다:

  ❌ 금지:
  가벼운 공감 중심 구조
  “좋아요 수” 기반 평가
  단순 리뷰 플랫폼화
  ⭕ 지향:
  긴장감 있는 상호작용
  지적 충돌
  사고의 깊이가 드러나는 구조
